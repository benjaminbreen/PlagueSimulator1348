# Plague Simulator: 1348

An experimental educational history game that simulates survival during the Black Death in medieval Damascus, Syria.

## About

**Plague Simulator: 1348** is a historically accurate, text-based survival simulation powered by Google Gemini AI. Players navigate the challenges of the 1348 plague outbreak through dynamic, AI-generated narratives while learning about medieval medicine, social structures, and the historical impact of the Black Death.

### Features

- **AI-Generated Storytelling**: Each playthrough is unique, with narratives, NPCs, and events generated by Gemini
- **Historical Accuracy**: Incorporates authentic medieval medical theory (humoral balance), social class dynamics, and period-appropriate challenges
- **ASCII Map Visualization**: Navigate medieval Damascus through procedurally generated environments
- **Character System**: Track health, symptoms, inventory, and family relationships
- **Interactive NPCs**: Engage in dialogue with dynamically generated historical characters
- **Medieval Medicine**: Experience 14th-century medical understanding through the humoral system

## Setup

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Google Gemini API key

### Installation

1. Clone the repository:
```bash
git clone https://github.com/benjaminbreen/PlagueSimulator1348.git
cd PlagueSimulator1348
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env.local` file in the root directory and add your Gemini API key:
```
VITE_GEMINI_API_KEY=your_api_key_here
```

   Get your API key from [Google AI Studio](https://ai.google.dev/)

4. Start the development server:
```bash
npm run dev
```

5. Open your browser to the URL shown in the terminal (typically `http://localhost:5173`)

## How to Play

### Controls

- **Arrow Keys**: Move around the map
- **Click NPCs/Items**: View detailed information
- **Custom Actions**: Type commands in the input field
- **Spacebar**: Interact with containers
- **Number Keys**: Select numbered options

### Game Mechanics

- **Health & Humors**: Monitor your character's wellbeing through medieval medical theory
- **Symptoms**: Track disease progression and respond accordingly
- **Inventory**: Collect and use items to survive
- **Social Class**: Your background affects available options and NPC interactions
- **Time**: Each action advances the clock; survival requires careful planning

### Tips

- Explore your surroundings carefully
- Pay attention to NPC dialogue for clues
- Manage your resources wisely
- Understanding medieval medicine concepts can help inform decisions
- Each playthrough generates a unique character and story

1. Push your code to GitHub (without the `.env.local` file - it's gitignored)

2. Import your repository in [Vercel](https://vercel.com)

3. Add the environment variable in Vercel:
   - Go to Project Settings â†’ Environment Variables
   - Add: `VITE_GEMINI_API_KEY` with your Gemini API key value
   - Make sure it's available for Production, Preview, and Development

4. Deploy! Vercel will automatically build and deploy your app

**Note**: The `VITE_` prefix is required for Vite to expose the variable to the client-side code.
